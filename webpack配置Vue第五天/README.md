- webpack
  + webpack 就是一个打包工具
  + webpack 把网页中的一切都认为是资源，一切资源都可以通过 JavaScript 来处理
  + 例如 css、img、less、sass、.vue 等都属于资源的一种
  + 它的核心目标就是让程序员越来越懒

- 从使用角度来讲
